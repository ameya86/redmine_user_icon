= User Icon

ユーザに自由なアイコンを付けることができます。

== インストール

1. RAILS_ROOT/vendor/pluginsに配置する。

2. Redmineを再起動する。


== アンインストール

1. RAILS_ROOT/vendor/pluginsから削除する。

2. Redmineを再起動する。

※アイコンはアンインストール前にすべて削除しなければ残ってしまいます。


== 注意事項

ユーザに関連付けされる添付ファイル機能を提供しているプラグインは考慮していないため、
同時に使用すると、アイコンのアップロード、削除時に他の添付ファイルを削除してしまいます。
